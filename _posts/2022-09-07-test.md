装饰器：
1. 将被装饰的函数传输装饰器函数
2. 执行装饰器函数，并将装饰器函数的返回值返回给被装饰函数对应的函数名
3. 整体的效果就是同时执行装饰器函数以及被装饰的函数

example1
```
def func1(func):
    print('a')
    return func()
    
@func1
def myprint():
    print('b')
    return 1
    
myprint
```

example2
```
def func1(func):
    def func2():
        print('a')
        return func()
    return func2
    
@func1
def myprint():
    print('b')
    
myprint
myprint()
```
